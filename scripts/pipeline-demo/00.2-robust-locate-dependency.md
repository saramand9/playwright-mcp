# 输入⑤：robustLocate 实现（最终脚本的运行时依赖）

## 来源

`D:\ai\claude_cli\templates\repair-helper.ts`

## 接口

```ts
function robustLocate(
  page: Page,
  options: {
    intent: string;                    // 人类可读的定位意图（用于 LLM 修复时的上下文）
    strategies: Array<{                // 定位策略数组，按优先级排列
      type: string;                    // 'semantic' | 'xpath' | 'fullXPath'
      locator: () => Locator;          // 定位器工厂函数
    }>;
    action: (locator: Locator) => Promise<void>;  // 定位成功后执行的断言或操作
    timeout: number;                   // 单层策略超时（ms）
  }
): Promise<void>
```

## 执行逻辑

```
strategies[0] → 尝试定位 → 成功? → action(locator) → 返回
                ↓ 失败
strategies[1] → 尝试定位 → 成功? → action(locator) → 返回
                ↓ 失败
strategies[2] → 尝试定位 → 成功? → action(locator) → 返回
                ↓ 失败
callLLMRepair(screenshot, snapshot, intent) → 修复后定位器 → action(locator)
```

## 使用示例

```ts
await robustLocate(page, {
  intent: 'Apple品牌筛选标签',
  strategies: [
    { type: 'semantic', locator: () => page.locator('#filter-panel').getByText('Apple') },
    { type: 'xpath', locator: () => page.locator('//*[@id="filter-panel"]//*[contains(text(),"Apple")]') },
    { type: 'fullXPath', locator: () => page.locator('/html/body/div[2]/div/div[2]/div/div[1]/div[1]') },
  ],
  action: (loc) => loc.click(),
  timeout: 5000,
});
```
